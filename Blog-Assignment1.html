Step-by-Step guide to learn Data Structure and Algorithms in 2025

00th 01st 02nd 03rd 04th 05th 06th 07th 08th 09th 10th 11th

0th Step

Before you start with your DSA journey, you first have to figure out the programming language you want to do DSA in.
Here are some suggesitons:

  1. C/C++
  2. Java
  3. Python
  4. JavaScript
After you've decided what language you want to proceed with, try to familiarize yourself with the language syntax by writting a few programs and then go on to learn some progaramming fundamentals like:
  • Pseudocodes
  • OOPs basics
  • Functions

1st Step

After your've got a hold of the language, get into the basics of what DSA really is. Don't panic! it's not a lot of theory; Just 2 questions:

  1. What are Data Structures?
  2. What is the importance of Data Structures?

2nd Step

Time to get started!
Now get into the basic data stuctures.

  1. Array
  2. Linked List
  3. Stacks
  4. Queues
  5. Hash Cookies Tables

3rd Step

The next thing to do now is to learn how to analyse these codes that you've been writting till now. As different users have various types of machines with varying hardware specifications, so we need a uniform way to analyse these programs for all the diffrent machines. And for that we have:

  • Time complexity
  • Space complexity
This topic is categorised as "Algorithmic Analysis" in books and is a fairly less time and effort, but don't think it is not an important part of DSA.
It wouldn't be wrong if somebody says that analysing the code is as important as being able to write that code.

4th Step

Now comes the "Algorithms" part of DSA. These can be divided into two types of algorithms:

  1. Sorting Algorithm
    • Bubble Sort
    • Merge Sort
    • Insertion Sort
    • Quick Sort
    • Selection Sort
    • Heap Sort
  2. Search Algorithms
    • Linear Search
    • Binary Search

5th Step

Now that you're done with Basic DSA, let's move on to Intermediate DSA concepts.
And the first one at that would be the Tree Data Structures and Algorithms. There are a few concepts in Tree:

  1. Types of Trees:
    • Binary Trees
    • Binary Search Trees
    • AVL Trees
    • B-Trees
  2. Tree Traversal Algorithms:
    • In-Order Traversal
    • Pre-Order Traversal
    • Post-Order Traversal
  3. Tree Search Algorithms:
    • Breadth First Search
    • Depth First Search

6th Step

The second step at the Intermediate level of DSA is Graph Data Structure and Algorithms, they include:

  1. Types of Graphs:
    • Directed Graph
    • Undirected Graph
  2. Graph Search Algorithms:
    • Breadth First Search
    • Depth First Search
  3. Shortest Path Algorithms:
    • Dijkstra's Algorithm
    • Bellman-Ford
  4. Minimum Spanning Tree:
    • Prim's Algorithm
    • Kruskal's Algorithm

7th Step

You've completed basic and intermediate DSA, now it's time to get into Advanced DSA concepts. Now you'll have to work on:

  1. Trie
  2. Segment Trees
  3. Ferwick Trees
  4. Disjoint Sets
  5. Suffic Trees and Arrays

8th Step

Let's now learn indexing which is a very important concept in Data Structures.

  1. Linear Indexing
  2. Tree-based indexing

9th Step

Congratulations! you are done with DSA. But wait there's more to DSA than just DSA.
There's DAA (Design & Analysis of Algorithms), it includes:

  1. Brute Force Strategy
  2. Backtracking
  3. Greedy Algorithm
  4. Randomised Algorithm
  5. Divide and Conquer
  6. Recursion
  7. Dynamic Programming
  8. Two-Pointer Technique
  9. Sliding window technique

10th Step

Great job on getting so far, only a few more steps to go. Alough what you've covered till now is enough to crack any placement round, but just in case you want to improve yourself more and stand out, here's what you need to do.
Complex Data Structures:

  1. B/B+ Trees
  2. Skip List
  3. ISAM
  4. 2-3 Trees

11th Step

Finally you are done with DSA completely and the only thing to do now is to:

  1. Practice
  2. Practice
  3. Practice
  4. ....
